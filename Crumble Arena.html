CRUMBLE ARENA

NEA PyGame Project

College NEA project. This game was built only using basic imports, including pygame. I made my own custom animation system using a spritesheet, where I also drew up my own assets.

The tilemap used a perlin noise texture, that was then filtered to output a square island shape. I then handled what texture from the spritesheet each tile would need to create the ocean, island edges and island itself. I created a system where tiles of the background tilemap would be unloaded and loaded depending on if they are on the screen.

I worked using Object Oriented Programming so then my code would maintain some structure and be more efficient, and inheriting to re-use code for new objects. I used a delta time system to make the game FPS independent. In the later stages of development I created some ui button elements to create a basic menu system, and then a loading screen to allow the game to load.

I never finished this project, but this project taught myself alot about time management, and misunderstanding how much work can be done in so much time.
